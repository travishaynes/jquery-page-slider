Bindings

After you set up your new PageSlider, you will need to bind it to some anchors to actually trigger the page-sliding magic.

bindTo (el, event)

The element that el describes must be an anchor with an href attribute present.

event is the event to bind to, which is usually a "click" event.

Example:

Bind to the click events of any anchors with a data-remote attribute set to true. 

<script type="text/javascript">
    var slider = new PageSlider();
    slider.bindTo("a[data-remote=true]", "click");
</script>
