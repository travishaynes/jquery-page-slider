JavaScript

Loading the library:

jQuery-page-slider depends on jQuery >= 1.7 to function. jQuery UI can be optionally added to use additional easing transitions.

Using the latest source from github:

Setting up a page slider:

<script type="text/javascript">
  $(document).ready(function(){
    // Ensure that we do not initialize the slider
    // more than once, since this code executes
    // on every page that slides in.
    if(!window.slider){
      // Create the slider
      window.slider = new PageSlider();
      // Bind to the click events of the
      // data-remote anchors on the page
      slider.bindTo("a[data-remote=true]", "click");
    }
  });
</script>

Using custom options:

var slider = new PageSlider({
  el        : "#content",
  easing    : "swing",
  duration  : 1200,
  tagCurrent: false
});

Binding to events:

// Bind to the mouseup events of button classes
slider.bindTo(".button", "mouseup");
