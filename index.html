jQuery Page Slider

jQuery Page Slider is a small (~2k) library that adds page sliding transitions to a website, using AJAX to dynamically load the content for the next page instead of having to refresh the entire document on every transition.

When the user has a modern browser that is compatible with the HTML5 history API, the page sliding transition works when the user clicks the browser's back / forward buttons, making the site function more like a web application than a website.
