CSS

The page transitions depend on CSS styling to function properly. The sliding transition works by animating a #current-page div from "left: 0%" to left: -100%", at the same time as animating a #next-page div from "left: 100%" to "left: 0%".

For convenience, the #current-page and #next-page divs are automatically inserted into the document when you create a new instance of PageSlider. Here are their inline CSS rules: 

#current-page {
  position: absolute;
  left    : 0;
  top     : 0;
  width   : 100%;
}
#next-page {
  position: absolute;
  left    : 100%;
  top     : 0;
  width   : 100%;
}

You will want to make sure that the parent element described in slider.el has a overflow: hidden; rule to prevent the page from appearing outside of the view pane, and the browser's scroll bars from appearing, allowing the user to scroll the #next-page div into focus.
