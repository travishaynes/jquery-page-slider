Sliding Content

slide (page)

Additional to loading the content via AJAX calls using the href of an anchor, you can also slide your own content into view by making a call to slider.slide(page); where page is a hash containing these required attributes: 

var page = {
  href    : "/example.html",
  title   : "Example Page",
  content : "<p>The HTML to slide into view.</p>"
}
