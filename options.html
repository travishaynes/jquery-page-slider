Options

When creating a new PageSlider you can pass a hash of options, or update them as properties of the slider after it has been created.

el

Default: "#main"

All your pages should have a common DOM element that contains the main content. el describes the jQuery selector for that element.

easing

Default: "linear"

A string indicating which easing function to use for the page slide transition. An easing function specifies the speed at which the animation progresses at different points within the animation.

The only easing implementations in this library are the jQuery defaults called "swing", and one that progresses at a constant pace, called "linear". More easing functions are available with the use of plug-ins, most notably the jQuery UI suite.

duration

Default: "fast"

A string or number determining how long the animation will run.

Duration is in milliseconds; higher values indicate a slower page slide transition, not a faster one. The strings "fast" and "slow" can be supplied to indicate durations of 200 and 600 milliseconds, respectively.

tagCurrent

Default: true

When set to true any anchor on the page that contains the current location.pathname will be tagged with a "current" class so you can style out the links for the current visible page.
